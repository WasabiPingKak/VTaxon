# VTaxon 更新公告（2026/3/15）

---

## 分類樹 — 視覺預算機制（全新功能）

複合種角色會同時出現在分類樹的多個位置，登記的物種越多，佔據的節點就越多。新增的**視覺預算機制**會根據 VTuber 登記的物種數量自動調整顯示等級，讓分類樹保持清晰。

### 自動分級顯示
- 一個 VTuber 登記的物種數量越多，在樹上的顯示等級越低：
  - **1~4 種**：正常顯示（頭像 + 名稱）
  - **5 種**：頭像以圓點取代
  - **6 種以上**：折疊隱藏，以「**+N 位**」badge 呈現，展開後仍以圓點取代頭像
- 點擊「+N 位」badge 可展開查看所有隱藏的 VTuber

### 代表物種豁免
- 你設定的**代表物種**不受視覺預算限制，至少會有一個節點永遠以正常等級顯示
- 沒有設定代表物種的話，會自動套用第一個物種作為代表

## SEO 優化

- 分享 VTuber 個人頁面的連結時，社群平台（Discord、Twitter 等）會正確顯示**名稱、頭像與簡介**的預覽卡片

## 體驗改善

- 所有面板的**卷軸樣式統一**，視覺更一致
- 手機版進階設定與篩選的**底部彈窗 UI 優化**
- 圖鑑計數在新增/刪除物種後**自動更新**，不再需要重新整理頁面
- 儲存個人資料後「有未儲存的變更」提示正確消失

- 修正多處節點間距與重疊問題，分類樹整體排版更整齊
