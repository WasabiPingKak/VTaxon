# VTaxon 更新公告（2026/3/14）

---

## 直播狀態偵測（全新功能）

VTaxon 現在可以**即時偵測 VTuber 的直播狀態**了！支援 Twitch 和 YouTube 雙平台。

### 分類樹上的直播提示
- 正在直播的 VTuber 節點會顯示 **LIVE 紅色徽章**（含脈動白點動畫）和紅色光暈
- 側邊欄新增「**直播中**」篩選按鈕，一鍵找到所有正在直播的人
- 篩選時會**自動展開**直播節點的樹路徑
- 新增**直播代表物種**設定：
  - 開啟篩選時，每人在現實與虛構物種的樹上各只會顯示**一個**節點
  - 複合種角色在樹上有多個節點，直播篩選時全部亮起會造成過多重複資訊
  - 設定了多個物種的使用者可以到個人頁面選擇要顯示哪一個，沒設定的話會自動套用第一個
- **節點配色重新設計**：當前使用者、直播中、一般節點各有獨立配色，視覺語意更明確

### 資訊卡與詳細面板
- 支援**雙平台同時直播**——YouTube 和 Twitch 同時開播時，兩個直播資訊並列顯示
- 開播時會自動抓取**直播標題**
- 點進 VTuber 詳細頁，直播中會顯示**頭像紅框 + LIVE 徽章 + 直播連結**
- 個人頁面也能看到目前的直播狀態

### 圖鑑頁面的直播整合
- 新增「**活躍優先**」排序：依直播結束後的時間排列，最近有直播活動的排在最前
- 所有排序模式下，直播中的 VTuber 都會**置頂顯示**
- 新增**直播優先開關**（Toggle Pill），可自由決定是否將直播中的角色排到最前面，預設為關閉
- 每張卡片會根據排序模式顯示對應的**動態時間 Badge**（「3分鐘前出沒」「2小時前出沒」「5天前出沒」等）

### 全自動啟用，不需額外操作
- 只要設定好頻道連結，系統就會**自動開始偵測你的直播狀態**，不需要額外設定
- 帳號設定頁的頻道卡片會顯示綠點「**直播通知已啟用**」，確認偵測已生效
- 萬一偵測啟用失敗，帳號設定頁會提示你，點一下「**重新連結**」即可修復
- YouTube 使用者如果登入時未授權頻道權限，帳號設定頁會顯示**引導說明**，幫助你完成授權
