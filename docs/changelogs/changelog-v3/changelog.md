# VTaxon 更新公告 (2026/3/12 ~ 3/14)

---

## 直播狀態偵測 (全新功能)

VTaxon 現在可以**即時偵測 VTuber 的直播狀態**了！支援 Twitch 和 YouTube 雙平台。

### 分類樹上的直播提示
- 正在直播的 VTuber 節點會顯示 **LIVE 紅色徽章** (含脈動白點動畫) 和紅色光暈
- 側邊欄新增 **"直播中" 篩選按鈕**，一鍵找到所有正在直播的人
- 篩選時會**自動展開**直播節點的樹路徑

### 資訊卡
- 支援**雙平台同時直播**——YouTube 和 Twitch 同時開播時，兩個直播資訊並列顯示
- 開播時會自動抓取**直播標題**

### VTuber 詳細面板
- 點進 VTuber 詳細頁，直播中會顯示**頭像紅框 + LIVE 徽章 + 直播連結**
- 個人頁面也能看到目前的直播狀態

### 直播訂閱狀態
- 設定頻道連結後，系統會**自動訂閱直播通知**，不需額外操作
- 訂閱成功時，頻道卡片會顯示 **綠點 "直播通知已啟用"**
- 如果訂閱失敗，帳號設定頁會顯示警告，提供 **"重新訂閱" 按鈕**一鍵修復

### YouTube 權限引導
- 如果 YouTube 權限尚未授權，帳號設定頁會顯示**引導說明**，幫助你完成授權

## 圖鑑頁面改進

### 新排序模式
- 新增 **"活躍優先" 排序**：依直播活動時間排列，最近有直播活動的排在最前
- 新增 **"組織" 排序**：依所屬組織分群
- 所有排序模式下，直播中的 VTuber 都會**置頂顯示**

### 直播優先 Toggle
- 新增**直播優先開關** (Toggle Pill)，可自由決定是否將直播中的角色排到最前面，預設為關閉

### 時間標籤
- 每張卡片會根據排序模式顯示對應的**動態時間 Badge** ("3分鐘前" "2小時前" "5天前" 等)

### 列表修正
- 組織欄允許換行，物種欄加上 tooltip，不再被截斷
